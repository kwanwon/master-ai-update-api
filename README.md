# Master AI Update API

자동 업데이트를 위한 API 서버

## 기능
- 버전 확인
- 업데이트 파일 다운로드
- 자동 업데이트 지원

## 보안
- HTTPS 지원
- 버전 검증
- 파일 무결성 검사

## 라이센스
이 프로젝트는 비공개 라이센스로 보호됩니다.

## 문의사항
버그 리포트나 기능 요청은 이슈 트래커를 사용해주세요.
