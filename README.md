# Master AI Update API

자동 업데이트를 위한 API 서버

## 설치 및 실행 방법
1. 필수 패키지 설치:
```bash
pip install -r requirements.txt
```

2. 서버 실행:
```bash
python app.py
```

## 기능
- 버전 확인
- 업데이트 파일 다운로드
- 자동 업데이트 지원

## 보안
- HTTPS 지원
- 버전 검증
- 파일 무결성 검사

## API 엔드포인트
- GET /: 서버 상태 확인
- GET /version: 현재 버전 확인
- GET /update: 업데이트 파일 다운로드
